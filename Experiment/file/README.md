### 文件说明
`dataset.py` 是数据集加载脚本

- `2018_leak_demand.csv`：2018年泄漏流量，列为管道，行为时间点
- `2019_leak_demand.csv`：2019年泄漏流量，列为管道，行为时间点
- `leak_demands.csv`：2018年和2019年泄漏流量合并，列为管道，行为时间点
- `no_leak_dataset_2018.pickle`：2018年无泄漏数据，包括管道流量、压力
- `no_leak_dataset_2019.pickle`：2019年无泄漏数据，包括管道流量、压力
- `leak_dataset_2018.pickle`：2018年有泄漏数据，包括管道流量、压力、流量、泄漏事件
- `leak_dataset_2019.pickle`：2019年有泄漏数据，包括管道流量、压力、流量、泄漏事件
- `fix_pressure_2018.pickle`：2018年修正后的压力
- `fix_pressure_2019.pickle`：2019年修正后的压力
- `fix_pressure.pickle`：2018年和2019年修正后的压力合并
- `fix_pressure.csv`：2018年和2019年修正后的压力合并，列为传感器，行为时间点
- `pressure_2018.pickle`：2018年原始压力
- `pressure_2019.pickle`：2019年原始压力
- `pressure.pickle`：2018年和2019年原始压力合并
- `pressure.csv`：2018年和2019年原始压力合并，列为传感器，行为时间点
- `PUMP_AB_model.pickle`：AB区域的水泵流量-传感器压力模型
- `PUMP_C_model.pickle`：C区域的水泵流量-传感器压力模型
- `PUMP_DMA_AB.pickle`：模拟AB区域泄漏，计算`PUMP_C_model`的流量-压力模型
- `PUMP_DMA_C.pickle`：模拟C区域泄漏，计算`PUMP_AB_model`的流量-压力模型
- `sensor_max.pickle`：硬分类
- `sensor_sort.pickle`：软分类
- `real_dataset_2018.pickle`：2018年真实数据，包括管道流量、压力
- `real_dataset_2019.pickle`：2019年真实数据，包括管道流量、压力
- `real_fix_pressure_2018.pickle`：2018年真实修正后的压力
- `real_fix_pressure_2019.pickle`：2019年真实修正后的压力
- `real_fix_pressure.pickle`：2018年和2019年真实修正后的压力合并
- `real_fix_pressure.csv`：2018年和2019年真实修正后的压力合并，列为传感器，行为时间点
